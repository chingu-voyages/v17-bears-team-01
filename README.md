## Requirements

Node.js [Link](https://nodejs.org/)

## Install
```
$ git clone https://github.com/chingu-voyages/v17-bears-team-01 
$ cd v17-bears-team-01 
$ npm run install-all
```
## Running 
```
$ npm run post-install
```
